## 📌 BOJ 12100 - 2048 (Easy)
- **문제 링크**: [백준 12100번](https://www.acmicpc.net/problem/12100)
- **풀이 요약**
  - 최초의 보드판을 깊은 복사하여 덱에 저장한다.
  - 덱에서 poll한 보드를 깊은 복사하여 상, 하, 좌, 우로 이동시킨다.
  - 각 이동 시 우선 0은 건너뛰고 같은 값이 있으면 *2로 저장. 이동 전 값은 0으로 초기화
  - 병합 후 중간에 빈 공간(0)을 상, 하, 좌, 우로 당긴다.
